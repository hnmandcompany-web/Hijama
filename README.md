# Hijama and More

Professional Hijama (cupping therapy) website built with Next.js 14, Tailwind CSS, and framer-motion.

**Domain:** hijamaandmore.com

## Stack

- Next.js 14 App Router
- TypeScript
- Tailwind CSS (custom design system)
- framer-motion (animations)
- lucide-react (icons)
- Netlify (deployment + forms)

## Development

```bash
npm install
npm run dev
```

## Deployment

Deploy to Netlify. The `netlify.toml` is pre-configured for the Next.js plugin.
